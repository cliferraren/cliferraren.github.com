Database Engineering

Climate Analysis & Exploration

Decided to treat myself for a long holiday vacation in Honolulu, Hawaii! To have a better insight in planning my trip, I decided to do some climate analysis on the area. After I have my weather analytical data, I have decide also to create an app and share to the community the weather data by providing a climate analysis api.

SQLAlchemy is to model my table schemas and created a SQLite database for my tables. I created table for measurements and one for stations. Development environment using Jupyter Notebook called database_engineering.ipynb and use it to complete all of my Database Engineering work.

Python Pandas was used to read my cleaned measurements and stations CSV data and then used the engine and connection string to create a database called hawaii.sqlite.

Declarative_base was used and created ORM classes for each table. Classes for Measurement and for Station were created with defined primary keys. Once I have my ORM classes defined, I created the tables in the database using create_all.
